开发环境设置

安装依赖：

npm install

启动开发模式：

npm run dev

启动生产模式：

npm start

构建

构建所有平台：npm run build

构建 Windows：npm run build:win

构建 macOS：npm run build:mac

构建 Linux：npm run build:linux

测试

npm test
